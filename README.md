# SOC Alert Investigation using ELK Stack

## 📌 Project Overview

This project simulates a real-world SOC Level 1 investigation using the ELK Stack (Elasticsearch, Logstash, Kibana).

## 🎯 Objectives

* Collect logs using Filebeat
* Analyze logs in Kibana
* Detect brute force attacks
* Create an incident report

## 🛠️ Tools Used

* Elasticsearch
* Kibana
* Filebeat

## 🔍 Scenario

Multiple failed login attempts were detected on a system. This project investigates whether it is a brute force attack.

## 📁 Project Structure

* setup-guide.md → Installation steps
* dashboards/ → Kibana visuals
* screenshots/ → Proof of work
* Incident-Reports/ → Investigation report

## ✅ Outcome

Successfully detected and analyzed suspicious login attempts using ELK.
